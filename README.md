## ChronoLog

ChronoLog now runs on a **React + Vite + Bun** stack with **shadcn/ui** components and **TanStack Router**.

### Prerequisites
- [Bun](https://bun.sh/) v1.1+

### Install
```bash
bun install
```

### Develop
```bash
bun run dev
```

### Build & Preview
```bash
bun run build
bun run preview
```

### Tech Stack
- React 18 + TypeScript with Vite bundling
- TanStack Router for SPA routing
- Tailwind CSS + shadcn/ui component primitives
- LocalStorage-backed auth/time entry helpers in `src/lib/storage.ts`
